\doxysubsection{Detailed Description}
A collection represents a group of objects.

Each object in the collection is of a specific, defined type.

This is a direct implementation of {\ttfamily Collection\+Interface}, provided for the sake of convenience.

Example usage\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{php}
\DoxyCodeLine{\ \ \$collection\ =\ new\ \(\backslash\)Ramsey\(\backslash\)Collection\(\backslash\)Collection(\textcolor{stringliteral}{'My\(\backslash\)\(\backslash\)Foo'});}
\DoxyCodeLine{\ \ \$collection-\/>add(\textcolor{keyword}{new}\ \(\backslash\)My\(\backslash\)Foo());}
\DoxyCodeLine{\ \ \$collection-\/>add(\textcolor{keyword}{new}\ \(\backslash\)My\(\backslash\)Foo());}
\DoxyCodeLine{\ }
\DoxyCodeLine{\ \ \textcolor{keywordflow}{foreach}\ (\$collection\ as\ \$foo)\ \{}
\DoxyCodeLine{\ \ \ \ \ \ \textcolor{comment}{//\ Do\ something\ with\ \$foo}}
\DoxyCodeLine{\ \ \}}

\end{DoxyCode}


It is preferable to subclass {\ttfamily Abstract\+Collection} to create your own typed collections. For example\+:
