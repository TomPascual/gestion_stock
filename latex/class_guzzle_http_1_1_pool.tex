\doxysubsection{Detailed Description}
Sends an iterator of requests concurrently using a capped pool size.

The pool will read from an iterator until it is cancelled or until the iterator is consumed. When a request is yielded, the request is sent after applying the "{}request\+\_\+options"{} request options (if provided in the ctor).

When a function is yielded by the iterator, the function is provided the "{}request\+\_\+options"{} array that should be merged on top of any existing options, and the function MUST then return a wait-\/able promise.
