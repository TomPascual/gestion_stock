\doxysubsection{Detailed Description}
Proc\+Output\+Pager class.

A Proc\+Output\+Pager instance wraps a regular Stream\+Output\textquotesingle{}s stream. Rather than writing directly to the stream, it shells out to a pager process and gives that process the stream as stdout. This means regular \texorpdfstring{$\ast$}{*}nix commands like {\ttfamily less} and {\ttfamily more} can be used to page large amounts of output.
