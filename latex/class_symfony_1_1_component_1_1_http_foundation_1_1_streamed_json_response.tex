\doxysubsection{Detailed Description}
Streamed\+Json\+Response represents a streamed HTTP response for JSON.

A Streamed\+Json\+Response uses a structure and generics to create an efficient resource-\/saving JSON response.

It is recommended to use flush() function after a specific number of items to directly stream the data.

flush()
\end{DoxySeeAlso}

Alexander Schranz \href{mailto:alexander@sulu.io}{\texttt{ alexander@sulu.\+io}}
\end{DoxyAuthor}
Example usage\+: \begin{DoxyVerb}function loadArticles(): \Generator
    // some streamed loading
    yield ['title' => 'Article 1'];
    yield ['title' => 'Article 2'];
    yield ['title' => 'Article 3'];
    // recommended to use flush() after every specific number of items
}),

$response = new StreamedJsonResponse(
    // json structure with generators in which will be streamed
    [
        '_embedded' => [
            'articles' => loadArticles(), // any generator which you want to stream as list of data
        ],
    ],
);
