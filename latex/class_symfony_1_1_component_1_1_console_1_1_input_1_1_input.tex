\doxysubsection{Detailed Description}
Input is the base class for all concrete Input classes.

Three concrete classes are provided by default\+:


\begin{DoxyItemize}
\item {\ttfamily Argv\+Input}\+: The input comes from the CLI arguments (argv)
\item {\ttfamily String\+Input}\+: The input is provided as a string
\item {\ttfamily Array\+Input}\+: The input is provided as an array
\end{DoxyItemize}
