\doxysubsection{Detailed Description}
This repository holds the interface for \href{https://www.php-fig.org/psr/psr-20}{\texttt{ PSR-\/20}}.

Note that this is not a clock of its own. It is merely an interface that describes a clock. See the specification for more details.\hypertarget{README.md_autotoc_md1603}{}\doxysubsection{\texorpdfstring{Installation}{Installation}}\label{README.md_autotoc_md1603}
