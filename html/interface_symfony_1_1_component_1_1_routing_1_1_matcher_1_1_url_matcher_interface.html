Symfony\Component\Routing\Matcher\UrlMatcherInterface::match ( string $pathinfo)

Tries to match a URL path with a set of routes.

If the matcher cannot find information, it must throw one of the exceptions documented below.

Parameters
string$pathinfoThe path info to be parsed (raw format, i.e. not urldecoded)
Exceptions
NoConfigurationExceptionIf no routing configuration could be found
ResourceNotFoundExceptionIf the resource could not be found
MethodNotAllowedExceptionIf the resource was found but the request method is not allowed
