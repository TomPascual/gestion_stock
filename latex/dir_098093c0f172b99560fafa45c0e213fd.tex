\doxysubsection{Detailed Description}
This component provides a partial, native PHP implementation of the \href{https://php.net/intl.grapheme}{\texttt{ Grapheme functions}} from the \href{https://php.net/intl}{\texttt{ Intl}} extension.


\begin{DoxyItemize}
\item \href{https://php.net/grapheme_extract}{\texttt{ {\ttfamily grapheme\+\_\+extract}}}\+: Extract a sequence of grapheme clusters from a text buffer, which must be encoded in UTF-\/8
\item \href{https://php.net/grapheme_stripos}{\texttt{ {\ttfamily grapheme\+\_\+stripos}}}\+: Find position (in grapheme units) of first occurrence of a case-\/insensitive string
\item \href{https://php.net/grapheme_stristr}{\texttt{ {\ttfamily grapheme\+\_\+stristr}}}\+: Returns part of haystack string from the first occurrence of case-\/insensitive needle to the end of haystack
\item \href{https://php.net/grapheme_strlen}{\texttt{ {\ttfamily grapheme\+\_\+strlen}}}\+: Get string length in grapheme units
\item \href{https://php.net/grapheme_strpos}{\texttt{ {\ttfamily grapheme\+\_\+strpos}}}\+: Find position (in grapheme units) of first occurrence of a string
\item \href{https://php.net/grapheme_strripos}{\texttt{ {\ttfamily grapheme\+\_\+strripos}}}\+: Find position (in grapheme units) of last occurrence of a case-\/insensitive string
\item \href{https://php.net/grapheme_strrpos}{\texttt{ {\ttfamily grapheme\+\_\+strrpos}}}\+: Find position (in grapheme units) of last occurrence of a string
\item \href{https://php.net/grapheme_strstr}{\texttt{ {\ttfamily grapheme\+\_\+strstr}}}\+: Returns part of haystack string from the first occurrence of needle to the end of haystack
\item \href{https://php.net/grapheme_substr}{\texttt{ {\ttfamily grapheme\+\_\+substr}}}\+: Return part of a string
\end{DoxyItemize}
