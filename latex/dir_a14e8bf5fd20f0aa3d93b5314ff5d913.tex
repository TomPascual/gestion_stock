\doxysubsection{Detailed Description}
Analyzes php-\/code for side-\/effects.

When code has no side-\/effects it can e.\+g. be used with {\ttfamily eval(\$code)} in the same process without interfering. \href{https://github.com/staabm/side-effects-detector/blob/main/lib/SideEffect.php}{\texttt{ Side-\/effects are classified}} into categories to filter them more easily depending on your use-\/case.\hypertarget{README.md_autotoc_md2260}{}\doxysubsubsection{\texorpdfstring{Install}{Install}}\label{README.md_autotoc_md2260}
