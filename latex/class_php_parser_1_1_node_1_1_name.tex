Concatenate two names, yielding a new Name instance.

The type of the generated instance depends on which class this method is called on, for example Name\textbackslash{}\+Fully\+Qualified\+::concat() will yield a Name\textbackslash{}\+Fully\+Qualified instance.

Gets a slice of a name (similar to array\+\_\+slice).

This method returns a new instance of the same type as the original and with the same attributes.

If the slice is empty, null is returned. The null value will be correctly handled in concatenations using concat().

Offset and length have the same meaning as in array\+\_\+slice().
