\doxysubsection{Detailed Description}
Sendmail\+Transport for sending mail through a Sendmail/\+Postfix (etc..) binary.

Transport can be instantiated through Sendmail\+Transport\+Factory or Native\+Transport\+Factory\+:


\begin{DoxyItemize}
\item Sendmail\+Transport\+Factory to use most common sendmail path and recommended options
\item Native\+Transport\+Factory when configuration is set via php.\+ini
\end{DoxyItemize}

Supported modes are -\/bs and -\/t, with any additional flags desired.

The recommended mode is "{}-\/bs"{} since it is interactive and failure notifications are hence possible. Note that the -\/t mode does not support error reporting and does not support Bcc properly (the Bcc headers are not removed).
