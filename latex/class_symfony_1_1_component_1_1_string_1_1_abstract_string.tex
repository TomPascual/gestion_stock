\doxysubsection{Detailed Description}
Represents a string of abstract characters.

Unicode defines 3 types of "{}characters"{} (bytes, code points and grapheme clusters). This class is the abstract type to use as a type-\/hint when the logic you want to implement doesn\textquotesingle{}t care about the exact variant it deals with.
