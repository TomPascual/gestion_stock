Creates a pretty printer instance using the given options.

Supported options\+:
\begin{DoxyItemize}
\item Php\+Version \$php\+Version\+: The PHP version to target (default to PHP 7.\+4). This option controls compatibility of the generated code with older PHP versions in cases where a simple stylistic choice exists (e.\+g. array() vs \mbox{[}\mbox{]}). It is safe to pretty-\/print an AST for a newer PHP version while specifying an older target (but the result will of course not be compatible with the older version in that case).
\item string \$newline\+: The newline style to use. Should be "{}\textbackslash{}n"{} (default) or "{}\textbackslash{}r\textbackslash{}n"{}.
\item string \$indent\+: The indentation to use. Should either be all spaces or a single tab. Defaults to four spaces ("{}    "{}).
\item bool \$short\+Array\+Syntax\+: Whether to use \mbox{[}\mbox{]} instead of array() as the default array syntax, if the node does not specify a format. Defaults to whether the php\+Version support short array syntax.
\end{DoxyItemize}
