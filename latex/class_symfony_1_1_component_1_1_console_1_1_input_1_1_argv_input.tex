\doxysubsection{Detailed Description}
Argv\+Input represents an input coming from the CLI arguments.

Usage\+: \begin{DoxyVerb}$input = new ArgvInput();
\end{DoxyVerb}


By default, the `\$\+\_\+\+SERVER\mbox{[}\textquotesingle{}argv'\mbox{]}\`{} array is used for the input values.

This can be overridden by explicitly passing the input values in the constructor\+: \begin{DoxyVerb}$input = new ArgvInput($_SERVER['argv']);
\end{DoxyVerb}


If you pass it yourself, don\textquotesingle{}t forget that the first element of the array is the name of the running application.

When passing an argument to the constructor, be sure that it respects the same rules as the argv one. It\textquotesingle{}s almost always better to use the {\ttfamily String\+Input} when you want to provide your own input.
