Returns a biased integer between \$min and \$max (both inclusive). The distribution depends on \$function.

The algorithm creates two doubles, x ∈ \mbox{[}0, 1\mbox{]}, y ∈ \mbox{[}0, 1) and checks whether the return value of \$function for x is greater than or equal to y. If this is the case the number is accepted and x is mapped to the appropriate integer between \$min and \$max. Otherwise two new doubles are created until the pair is accepted.
