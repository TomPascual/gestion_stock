Guzzle\+Http\textbackslash{}\+Psr7\textbackslash{}\+Response\+::get\+Reason\+Phrase (\begin{DoxyParamCaption}{}{}\end{DoxyParamCaption})}

Gets the response reason phrase associated with the status code.

Because a reason phrase is not a required element in a response status line, the reason phrase value MAY be null. Implementations MAY choose to return the default RFC 7231 recommended reason phrase (or those listed in the IANA HTTP Status Code Registry) for the response\textquotesingle{}s status code.

\doxylink{}{\href{http://www.iana.org/assignments/http-status-codes/http-status-codes.xhtml}{\texttt{ http\+://www.\+iana.\+org/assignments/http-\/status-\/codes/http-\/status-\/codes.\+xhtml}}  string Reason phrase; must return an empty string if none present. }
