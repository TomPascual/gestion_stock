\begin{DoxyItemize}
\item A {\ttfamily Test\textbackslash{}Preparation\+Failed} event is now emitted in addition to a {\ttfamily Test\textbackslash{}Errored} event when an unexpected exception is triggered in a before-\/test method
\item A {\ttfamily Test\textbackslash{}Passed} event is no longer emitted in addition to a {\ttfamily Test\textbackslash{}Failed} or {\ttfamily Test\textbackslash{}Errored} event when an assertion failure or an unexpected exception is triggered in an after-\/test method ~\newline

\item A {\ttfamily Test\+Suite\textbackslash{}Finished} event is now emitted when a before-\/first-\/test method errors
