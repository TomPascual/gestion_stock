\doxysubsection{Detailed Description}
Nonstandard UUID fields do not conform to the RFC 4122 standard

Since some systems may create nonstandard UUIDs, this implements the Rfc4122\textbackslash{}\+Fields\+Interface, so that functionality of a nonstandard UUID is not degraded, in the event these UUIDs are expected to contain RFC 4122 fields.

Internally, this class represents the fields together as a 16-\/byte binary string.

Returns the version

The version number describes how the UUID was generated and has the following meaning\+:


\begin{DoxyEnumerate}
\item Gregorian time UUID
\item DCE security UUID
\item Name-\/based UUID hashed with MD5
\item Randomly generated UUID
\item Name-\/based UUID hashed with SHA-\/1
\item Reordered time UUID
\item Unix Epoch time UUID
\end{DoxyEnumerate}

This returns {\ttfamily null} if the UUID is not an RFC 4122 variant, since version is only meaningful for this variant.
