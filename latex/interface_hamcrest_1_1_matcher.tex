\doxysubsection{Detailed Description}
A matcher over acceptable values. A matcher is able to describe itself to give feedback when it fails. 

Matcher implementations should {\bfseries{NOT directly implement this interface}}. Instead, {\bfseries{extend}} the {\ttfamily \doxylink{}{Hamcrest\textbackslash{}\+Base\+Matcher}} abstract class, which will ensure that the Matcher API can grow to support new features and remain compatible with all Matcher implementations. 

For easy access to common Matcher implementations, use the static factory methods in {\ttfamily \doxylink{}{Hamcrest\textbackslash{}\+Core\+Matchers}}.
