Symfony\textbackslash{}\+Component\textbackslash{}\+Mime\textbackslash{}\+Address\+::has\+Unicode\+Localpart (\begin{DoxyParamCaption}{}{}\end{DoxyParamCaption})}

Returns true if this address\textquotesingle{} localpart contains at least one non-\/\+ASCII character, and false if it is only ASCII (or empty).

This is a helper for Envelope, which has to decide whether to the SMTPUTF8 extensions (RFC 6530 and following) for any given message.

The SMTPUTF8 extension is strictly required if any address contains a non-\/\+ASCII character in its localpart. If non-\/\+ASCII is only used in domains (e.\+g. \href{mailto:horst@freiherr-von-m\%C3\%BChlhausen.de}{\texttt{ horst@freiherr-\/von-\/m√ºhlhausen.\+de}}) then it is possible to send the message using IDN encoding instead of SMTPUTF8. The most common software will display the message as intended.
