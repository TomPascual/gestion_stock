Combines an array of arrays into one associative array.

Each of the nested arrays should have one or two elements. The first value will be used as the keys in the associative array, and the second will be used as the values, or true if the nested array only contains one element. Array keys are lowercased.

Example\+: \begin{DoxyVerb}HeaderUtils::combine([['foo', 'abc'], ['bar']])
// => ['foo' => 'abc', 'bar' => true]

Joins an associative array into a string for use in an HTTP header.

The key and value of each entry are joined with \textquotesingle{}=\textquotesingle{}, and all entries are joined with the specified separator and an additional space (for readability). Values are quoted if necessary.

Example\+: \begin{DoxyVerb}HeaderUtils::toString(['foo' => 'abc', 'bar' => true, 'baz' => 'a b c'], ',')
// => 'foo=abc, bar, baz="a b c"'
\end{DoxyVerb}
