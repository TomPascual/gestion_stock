\doxysubsection{Detailed Description}
Validate that the constructor method is not static, and does not have a return type.

Checks both explicit \+\_\+\+\_\+construct methods as well as old-\/style constructor methods with the same name as the class (for non-\/namespaced classes).

As of PHP 5.\+3.\+3, methods with the same name as the last element of a namespaced class name will no longer be treated as constructor. This change doesn\textquotesingle{}t affect non-\/namespaced classes.
