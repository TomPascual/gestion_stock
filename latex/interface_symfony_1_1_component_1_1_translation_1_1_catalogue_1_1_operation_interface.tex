\doxysubsection{Detailed Description}
Represents an operation on catalogue(s).

An instance of this interface performs an operation on one or more catalogues and stores intermediate and final results of the operation.

The first catalogue in its argument(s) is called the \textquotesingle{}source catalogue\textquotesingle{} or \textquotesingle{}source\textquotesingle{} and the following results are stored\+:

Messages\+: also called \textquotesingle{}all\textquotesingle{}, are valid messages for the given domain after the operation is performed.

New Messages\+: also called \textquotesingle{}new\textquotesingle{} (new = all ∖ source = \{x\+: x ∈ all ∧ x ∉ source\}).

Obsolete Messages\+: also called \textquotesingle{}obsolete\textquotesingle{} (obsolete = source ∖ all = \{x\+: x ∈ source ∧ x ∉ all\}).

Result\+: also called \textquotesingle{}result\textquotesingle{}, is the resulting catalogue for the given domain that holds the same messages as \textquotesingle{}all\textquotesingle{}.
