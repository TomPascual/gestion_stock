Filters an array of files.

This method created test instances of Uploaded\+File so that the move() method can be called on those instances.

If the size of a file is greater than the allowed size (from php.\+ini) then an invalid Uploaded\+File is returned with an error set to UPLOAD\+\_\+\+ERR\+\_\+\+INI\+\_\+\+SIZE.
