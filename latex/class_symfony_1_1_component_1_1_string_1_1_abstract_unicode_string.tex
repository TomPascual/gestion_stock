\doxysubsection{Detailed Description}
Represents a string of abstract Unicode characters.

Unicode defines 3 types of "{}characters"{} (bytes, code points and grapheme clusters). This class is the abstract type to use as a type-\/hint when the logic you want to implement is Unicode-\/aware but doesn\textquotesingle{}t care about code points vs grapheme clusters.
