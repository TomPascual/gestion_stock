Detailed Description

PHP getallheaders() polyfill. Compatible with PHP >= 5.3.

This is a simple polyfill for getallheaders().

Install

For PHP version >= 5.6:

composer require ralouphie/getallheaders

For PHP version < 5.6:

composer require ralouphie/getallheaders "^2"
