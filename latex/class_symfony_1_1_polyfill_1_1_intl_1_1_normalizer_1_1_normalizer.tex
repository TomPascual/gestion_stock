\doxylink{class_normalizer}{Normalizer} is a PHP fallback implementation of the \doxylink{class_normalizer}{Normalizer} class provided by the intl extension.

It has been validated with Unicode 6.\+3 Normalization Conformance Test. See \href{http://www.unicode.org/reports/tr15/}{\texttt{ http\+://www.\+unicode.\+org/reports/tr15/}} for detailed info about Unicode normalizations.
