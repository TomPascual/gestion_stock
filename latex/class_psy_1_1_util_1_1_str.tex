Decodes a string encoded by libsd\textquotesingle{}s strvis.

From {\ttfamily man 3 vis}\+:

Use an ‘\+M’ to represent meta characters (characters with the 8th bit set), and use a caret ‘\texorpdfstring{$^\wedge$}{\string^}’ to represent control characters (see iscntrl(3)). The following formats are used\+: \begin{DoxyVerb} \040    Represents ASCII space.

 \240    Represents Meta-space (&nbsp in HTML).

 \M-C    Represents character ‘C’ with the 8th bit set.
         Spans characters ‘\241’ through ‘\376’.

 \M^C    Represents control character ‘C’ with the 8th bit set.
         Spans characters ‘\200’ through ‘\237’, and ‘\377’ (as ‘\M^?’).

 \^C     Represents the control character ‘C’.
         Spans characters ‘\000’ through ‘\037’, and ‘\177’ (as ‘\^?’).
\end{DoxyVerb}


The other formats are supported by PHP\textquotesingle{}s stripcslashes, except for the \textbackslash{}s sequence (ASCII space).
