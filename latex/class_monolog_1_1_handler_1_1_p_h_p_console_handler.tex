\doxysubsection{Detailed Description}
Monolog handler for Google Chrome extension "{}\+PHP Console"{}

Display PHP error/debug log messages in Google Chrome console and notification popups, executes PHP code remotely

Usage\+:
\begin{DoxyEnumerate}
\item Install Google Chrome extension \mbox{[}now dead and removed from the chrome store\mbox{]}
\item See overview \href{https://github.com/barbushin/php-console\#overview}{\texttt{ https\+://github.\+com/barbushin/php-\/console\#overview}}
\item Install PHP Console library \href{https://github.com/barbushin/php-console\#installation}{\texttt{ https\+://github.\+com/barbushin/php-\/console\#installation}}
\item Example (result will looks like \href{http://i.hizliresim.com/vg3Pz4.png}{\texttt{ http\+://i.\+hizliresim.\+com/vg3\+Pz4.\+png}})
