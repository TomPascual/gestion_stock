\doxysubsection{Detailed Description}
\href{https://coveralls.io/github/nikic/PHP-Parser?branch=master}{\texttt{ }}

This is a PHP parser written in PHP. Its purpose is to simplify static code analysis and manipulation.

\href{https://github.com/nikic/PHP-Parser/tree/master/doc}{\texttt{ {\bfseries{Documentation for version 5.\+x}}}} (current; for running on PHP \texorpdfstring{$>$}{>}= 7.\+4; for parsing PHP 7.\+0 to PHP 8.\+4, with limited support for parsing PHP 5.\+x).

\href{https://github.com/nikic/PHP-Parser/tree/4.x/doc}{\texttt{ Documentation for version 4.\+x}} (supported; for running on PHP \texorpdfstring{$>$}{>}= 7.\+0; for parsing PHP 5.\+2 to PHP 8.\+3).\hypertarget{README.md_autotoc_md1327}{}\doxysubsection{\texorpdfstring{Features}{Features}}\label{README.md_autotoc_md1327}
The main features provided by this library are\+:


\begin{DoxyItemize}
\item Parsing PHP 7, and PHP 8 code into an abstract syntax tree (AST).
\begin{DoxyItemize}
\item Invalid code can be parsed into a partial AST.
\item The AST contains accurate location information.
\end{DoxyItemize}
\item Dumping the AST in human-\/readable form.
\item Converting an AST back to PHP code.
\begin{DoxyItemize}
\item Formatting can be preserved for partially changed ASTs.
\end{DoxyItemize}
\item Infrastructure to traverse and modify ASTs.
\item Resolution of namespaced names.
\item Evaluation of constant expressions.
\item Builders to simplify AST construction for code generation.
\item Converting an AST into JSON and back.
\end{DoxyItemize}\hypertarget{README.md_autotoc_md1328}{}\doxysubsection{\texorpdfstring{Quick Start}{Quick Start}}\label{README.md_autotoc_md1328}
Install the library using \href{https://getcomposer.org}{\texttt{ composer}}\+: \begin{DoxyVerb}php composer.phar require nikic/php-parser
