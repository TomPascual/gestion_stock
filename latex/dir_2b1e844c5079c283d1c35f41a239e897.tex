Sail provides a Docker powered local development experience for Laravel that is compatible with mac\+OS, Windows (WSL2), and Linux. Other than Docker, no software or libraries are required to be installed on your local computer before using Sail. Sail\textquotesingle{}s simple CLI means you can start building your Laravel application without any previous Docker experience.\hypertarget{README.md_autotoc_md900}{}\doxysubsubsubsubsection{\texorpdfstring{Inspiration}{Inspiration}}\label{README.md_autotoc_md900}
