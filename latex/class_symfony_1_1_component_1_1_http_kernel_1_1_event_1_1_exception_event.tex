\doxysubsection{Detailed Description}
Allows to create a response for a thrown exception.

Call set\+Response() to set the response that will be returned for the current request. The propagation of this event is stopped as soon as a response is set.

You can also call set\+Throwable() to replace the thrown exception. This exception will be thrown if no response is set during processing of this event.
