\doxysubsection{Detailed Description}
The UID component provides an object-\/oriented API to generate and represent UIDs.

It provides implementations that work on 32-\/bit and 64-\/bit CPUs for ULIDs and for UUIDs version 1 and versions 3 to 8.\hypertarget{README.md_autotoc_md2542}{}\doxysubsection{\texorpdfstring{Resources}{Resources}}\label{README.md_autotoc_md2542}
