\doxysubsection{Detailed Description}
Sends errors to Rollbar

If the context data contains a {\ttfamily payload} key, that is used as an array of payload options to Rollbar\+Logger\textquotesingle{}s log method.

Rollbar\textquotesingle{}s context info will contain the context + extra keys from the log record merged, and then on top of that a few keys\+:


\begin{DoxyItemize}
\item level (rollbar level name)
\item monolog\+\_\+level (monolog level name, raw level, as rollbar only has 5 but monolog 8)
\item channel
\item datetime (unix timestamp)
\end{DoxyItemize}
