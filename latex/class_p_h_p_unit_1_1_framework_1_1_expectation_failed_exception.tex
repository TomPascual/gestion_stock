\doxysubsection{Detailed Description}
Exception for expectations which failed their check.

The exception contains the error message and optionally a Sebastian\+Bergmann\textbackslash{}\+Comparator\textbackslash{}\+Comparison\+Failure which is used to generate diff output of the failed expectations.
