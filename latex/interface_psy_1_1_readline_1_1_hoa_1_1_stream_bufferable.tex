\doxysubsection{Detailed Description}
Interface \textbackslash{}\+Hoa\textbackslash{}\+Stream\textbackslash{}\+IStream\textbackslash{}\+Bufferable.

Interface for bufferable streams. It\textquotesingle{}s complementary to native buffer support of Hoa\textbackslash{}\+Stream (please, see {\itshape Stream\+Buffer}() methods). Classes implementing this interface are able to create nested buffers, flush them etc.
