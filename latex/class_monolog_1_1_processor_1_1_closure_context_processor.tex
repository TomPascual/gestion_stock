\doxysubsection{Detailed Description}
Generates a context from a Closure if the Closure is the only value in the context

It helps reduce the performance impact of debug logs if they do need to create lots of context information. If this processor is added on the correct handler the context data will only be generated when the logs are actually logged to that handler, which is useful when using Fingers\+Crossed\+Handler or other filtering handlers to conditionally log records.
