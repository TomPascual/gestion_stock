\texorpdfstring{$>$}{>}{\bfseries{Note}} This is the {\bfseries{documentation for Monolog 3.\+x}}, if you are using older releases \texorpdfstring{$>$}{>}see the documentation for \href{https://github.com/Seldaek/monolog/blob/2.x/README.md}{\texttt{ Monolog 2.\+x}} or \href{https://github.com/Seldaek/monolog/blob/1.x/README.md}{\texttt{ Monolog 1.\+x}}

Monolog sends your logs to files, sockets, inboxes, databases and various web services. See the complete list of handlers below. Special handlers allow you to build advanced logging strategies.

This library implements the \href{https://github.com/php-fig/fig-standards/blob/master/accepted/PSR-3-logger-interface.md}{\texttt{ PSR-\/3}} interface that you can type-\/hint against in your own libraries to keep a maximum of interoperability. You can also use it in your applications to make sure you can always use another compatible logger at a later time. As of 1.\+11.\+0 Monolog public APIs will also accept PSR-\/3 log levels. Internally Monolog still uses its own level scheme since it predates PSR-\/3.
