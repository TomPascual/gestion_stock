Sebastian\+Bergmann\textbackslash{}\+Environment\textbackslash{}\+Runtime\+::get\+Current\+Settings (\begin{DoxyParamCaption}\item[{array}]{\$values}{}\end{DoxyParamCaption})}

Parses the loaded php.\+ini file (if any) as well as all additional php.\+ini files from the additional ini dir for a list of all configuration settings loaded from files at startup. Then checks for each php.\+ini setting passed via the {\ttfamily \$values} parameter whether this setting has been changed at runtime. Returns an array of strings where each string has the format {\ttfamily key=value} denoting the name of a changed php.\+ini setting with its new value.
