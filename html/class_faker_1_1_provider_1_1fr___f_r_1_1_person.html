Generates a NIR / Sécurité Sociale number (13 digits + 2 digits for the key)

See also
https://fr.wikipedia.org/wiki/Num%C3%A9ro_de_s%C3%A9curit%C3%A9_sociale_en_France
Returns
string

The key for a given NIR is 97 - 97 % NIR NIR has to be an integer, so we have to do a little replacment for departments 2A and 2B
