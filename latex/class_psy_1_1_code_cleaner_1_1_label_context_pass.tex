\doxysubsection{Detailed Description}
Code\+Cleaner\+Pass for label context.

This class partially emulates the PHP label specification. Psy\+SH can not declare labels by sequentially executing lines with eval, but since it is not a syntax error, no error is raised. This class warns before invalid goto causes a fatal error. Since this is a simple checker, it does not block real fatal error with complex syntax. (ex. it does not parse inside function.)
