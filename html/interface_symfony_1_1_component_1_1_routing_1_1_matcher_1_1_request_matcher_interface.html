Symfony\Component\Routing\Matcher\RequestMatcherInterface::matchRequest ( Request $request)

Tries to match a request with a set of routes.

If the matcher cannot find information, it must throw one of the exceptions documented below.

Exceptions
NoConfigurationExceptionIf no routing configuration could be found
ResourceNotFoundExceptionIf no matching resource could be found
MethodNotAllowedExceptionIf a matching resource was found but the request method is not allowed
